# webchat

### description

text chat

### usage

##### environment
* redis 2.8.17
* python 3.5.3
* node 5.5.0

##### install

* `pip install -r requirements.txt`
* `npm install`

##### build and run

* `node bin/build_client.js`
* `PORT=5005 python bin/run.py`
  - replace `5005` with alternate port number
